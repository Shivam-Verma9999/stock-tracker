Its an angular app which tracks stock info in real-time (updated every 5 min).

This app uses non-prime APi  so it limits the number of requests. 5 requests per minute.



==============================  Deployment Instructions ===============

run "npm install " to install all dependencies
then run "npm start".
This will start server on "http://localhost:4200"

Requires Internet.


====  to build 
	run "ng build". This will create a dist folder which can be hosted anywhere to make it globally.
